# AscendAI

Premium SwiftUI MVP for a dark-mode facial analysis experience.

Included in this version:

- Onboarding
- Photo upload with `PhotosUI`
- Questionnaire
- Scanning animation
- Analysis steps
- Local fallback mock report
- Paywall placeholder

Not included yet:

- Gemini API
- Real AI analysis
- StoreKit
- Third-party libraries
